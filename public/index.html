Setup

Up and running in 2 minutes.

Add to your AI agent config and restart. No API keys required for the free tier.

{
  "mcpServers": {
    "multichain": {
      "command": "npx",
      "args": ["-y", "@wkalidev/multichain-mcp"],
      "env": {
        "MULTICHAIN_LICENSE_KEY": "your-key"
      }
    }
  }
}

Add this to your claude_desktop_config.json, then restart Claude Desktop.

{
  "mcpServers": {
    "multichain": {
      "command": "npx",
      "args": ["-y", "@wkalidev/multichain-mcp"],
      "env": {
        "MULTICHAIN_LICENSE_KEY": "your-key"
      }
    }
  }
}

Add this to .cursor/mcp.json or .windsurf/mcp.json.
